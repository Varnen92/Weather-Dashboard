# Weather-Dashboard

## Purpose
A showcase of a weather Dashboard that uses the given city to display current day data as well as a five day forecast. Displays searches during session to retrieve for quick reference back.

## Built with
* HTML
* CSS
* Bootstrap
* JavaScript
